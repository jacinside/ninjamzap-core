## Based on NINJAM

**NINJAM** (Novel Intervallic Network Audio Monitor)
**Copyright (C) 2004-2025 Cockos Incorporated**
**Created by**: Justin Frankel

Original NINJAM source:
- https://github.com/justinfrankel/ninjam
- https://ninjam.com

NINJAM is a revolutionary system for collaborative music-making over the internet. Musicians can jam in real-time, with the system handling network latency through synchronized intervals.

## Derived from abNinjam

**abNinjam** - iOS NINJAM Client Implementation
**Copyright (C) Antanas Bruzas**
**License**: GPL (inherited from NINJAM)
**Source**: https://github.com/antanasbruzas/abNinjam

This project's iOS wrapper and C++ integration is based on or inspired by abNinjam, an excellent iOS implementation of the NINJAM protocol. The `NinjamClientBridge` maintains this heritage while extending support for cross-platform mobile development.

Special thanks to **Antanas Bruzas** for pioneering NINJAM on iOS and making the code available to the community.
