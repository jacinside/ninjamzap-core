# Android Support (Coming Soon)

Android support for NinjamZap Core is planned but not yet implemented.

The C++ core is ready and cross-platform. Implementation requires a JNI wrapper in `platforms/android/jni/`.

- [JNI Documentation](https://docs.oracle.com/javase/8/docs/technotes/guides/jni/)
